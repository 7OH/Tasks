Инструкция по настройке уведомлений по аське. 


Бот ICQ позволяет рассылать уведомления по списку или отсылать одиночные сообщения. 

Для работы и уведомлений и бота достаточно чтобы сервер 1С имел доступ в интернет. Опубликовывать базу не надо. 

Для работы необходим Общий модуль КоннекторHTTP. Огромное спасибо Vladimir Bondarevskiy за него.

Информация по константам и модулям:
  • В справочник Пользователи появилось два новых реквизита: szНомерАськи, szПолучатьРассылкуПоАське.
  • В справочнике Пользователи необходимо заполнить реквизит szНомерАськи. 
  • Установить токен бота в обработке SZ Демо Аська. Без этого рассылка уведомлений работать не будет.
ВАЖНО. Бот может отсылать сообщения только тем пользователям, которые писали ему. Т.е. чтобы получать уведомления пользователь должен в аське написать боту хоть что-то. 

Для получения токена для вашего бота его необходимо создать. 
Для этого:
  • обратитесь к Метаботу
  • введите команду /newbot
  • придумайте и введите уникальное имя для своего бота. Имя обязательно должно оканчиваться на bot.
Если вы все сделали правильно, то вы получите сообщение Success и токен вашего бота.

Надо иметь в виду, что при отправке одиночного сообщения признак szПолучатьРассылкуПоАське у Пользователя игнорируется. Т.е. если номер аськи прописан, то Пользователь получит сообщение.
